## FOR QBITTORRENT ONLY::
All it does is takes `.torrent`'s from a folder on your pc and adds them to qbit. 

`pip install requests bencode.py`
Also enable webui and config that with a user and pass in qbit.
Put in your creds in the script.
Then it'll just grab the torrents from x folder and download them, correctly sort them and name them in qbit so you don't just have 1000 torrents all going that just are named "Minerva_Myrient"
